Error

public enum Error

A set of errors that may be encountered at run-time when interacting with a recored HTTP interaction.

  • recordNotFound

    During VCR playback an attempt was made to match the URLRequest provided in the associated value. However, no Track could be found that matched the request.

    Note

    This error is only expected when using a custom matcher.

    Declaration

    Swift

    case recordNotFound(for: URLRequest)
  • missing

    The VHS library was unable to find the resource provided in the associated value while attempting to load a Cassette fixture.

    Declaration

    Swift

    case missing(resource: String)
  • invalidFormat

    The fixture resource provided in the associated value is improperly formatted.

    Declaration

    Swift

    case invalidFormat(resource: String)
